After you finish setting up your first Brains entry, you do not need to configure other channels right away. You can start chatting in the web UI first.

Click **Chat**.

<Frame>

</Frame>

Send a message to test the setup.

<Frame>
  ![Image](/images/image-9.png)
</Frame>

Aurogen uses the same bootstrap mechanism as OpenClaw, so you can initialize your bot with a custom setup.

The **Event Timeline** in the lower-right corner shows the thinking and tool call activity.
